# nacalui

個人プロジェクト全般で一貫したユーザー体験を提供するためのデザインシステムです。

## Design Language

- **[Vision](?path=/docs/design-language-vision--docs)** — デザインシステムの目的と方向性
- **[Principles](?path=/docs/design-language-principles--docs)** — アクセシビリティ・一貫性・静けさ
- **[Accessibility](?path=/docs/design-language-accessibility--docs)** — WCAG 2.1 AA 準拠のガイドライン
- **[Terminology](?path=/docs/design-language-terminology--docs)** — 用語の定義
- **[Writing](?path=/docs/design-language-writing--docs)** — UIテキストの書き方

## Foundations

- **Colors** — 落ち着いたウォームニュートラルベースのカラーパレット
- **Typography** — Inter + 日本語フォントのタイプスケール
- **Spacing** — 4pxベースのスペーシングスケール
- **Elevation** — ウォームトーンのシャドウシステム
- **Radius** — 角丸トークン
- **Motion** — イージング・デュレーショントークン
