# Iconography

nacalui では UnoCSS の `preset-icons` を通じて [Iconify](https://iconify.design/) のアイコンを使います。
CSS クラスだけでアイコンを表示でき、バンドルサイズへの影響も最小限です。

---

## アイコンセット

**[Lucide](https://lucide.dev/)** をプライマリアイコンセットとして採用します。

- シンプルで落ち着いたストロークスタイル — nacalui のトーンと合う
- 一貫した 24x24 グリッド、2px ストローク
- MIT ライセンス
- 1000+ のアイコンを収録

Lucide で足りない場合のみ、他のセットから補完します。

## 使い方

```html
<!-- 基本 -->
<div class="i-lucide-search" />

<!-- サイズ変更 -->
<div class="i-lucide-search text-lg" />
<div class="i-lucide-search text-2xl" />

<!-- 色を変更 -->
<div class="i-lucide-search text-blue-600" />
<div class="i-lucide-search text-semantic-fg-primary" />

<!-- ボタン内 -->
<button class="flex items-center gap-2">
  <div class="i-lucide-plus" />
  追加する
</button>
```

## サイズ規約

- **インラインテキスト** — `text-sm`（14px）
- **デフォルト（ボタン等）** — `text-base`（16px）
- **強調・ナビゲーション** — `text-xl`（20px）
- **大きいアイコン** — `text-2xl`（24px）

## スタイルルール

- **ストロークスタイルのみ** — 塗りつぶし（filled）アイコンは使わない
- **同じセットから** — 1つの画面内で複数のアイコンセットを混在させない
- **意味のあるアイコンにはラベルを** — アイコン単体のボタンには `aria-label` を必ず付ける
- **装飾アイコンは `aria-hidden`** — テキストと併用する場合、アイコンは `aria-hidden="true"`

## アクセシビリティ

```html
<!-- アイコン + テキスト → アイコンは装飾 -->
<button>
  <div class="i-lucide-trash-2" aria-hidden="true" />
  削除する
</button>

<!-- アイコンのみ → aria-label 必須 -->
<button aria-label="削除する">
  <div class="i-lucide-trash-2" />
</button>
```

## 予約アイコン

以下のアイコンは特定の意味に固定して使います。一貫性のために他の用途で使わないでください。

- ✕ `i-lucide-x` — 閉じる・キャンセル
- ← `i-lucide-arrow-left` — 戻る
- ✓ `i-lucide-check` — 完了・成功
- ⚠ `i-lucide-alert-triangle` — 警告
- ℹ `i-lucide-info` — 情報
- ⊘ `i-lucide-alert-circle` — エラー
- ⋯ `i-lucide-more-horizontal` — その他のアクション
- 🔍 `i-lucide-search` — 検索
- ＋ `i-lucide-plus` — 追加・新規作成
- 🗑 `i-lucide-trash-2` — 削除
