# Accessibility Guidelines

nacalui は WCAG 2.1 AA 準拠を目標とします。

---

## カラー

- **テキスト:** 通常テキスト（16px未満）はコントラスト比 **4.5:1** 以上
- **大きいテキスト:** 18px以上または14px太字はコントラスト比 **3:1** 以上
- **UIコンポーネント:** ボタン境界線、フォーム入力枠などはコントラスト比 **3:1** 以上
- **色だけに頼らない:** エラーは赤色＋アイコン＋テキストで伝える

## キーボード

- すべてのインタラクティブ要素に `Tab` でフォーカスできる
- フォーカスインジケーターは常に視認できる（`outline` を消さない）
- モーダルはフォーカストラップを実装する
- `Escape` で閉じられるものは `Escape` で閉じる
- カスタムコンポーネントは適切な WAI-ARIA パターンに従う

## スクリーンリーダー

- 意味のある画像には代替テキストを付ける
- アイコンのみのボタンには `aria-label` を付ける
- ライブリージョン（`aria-live`）でトーストや動的な変更を通知する
- 見出しの階層構造を正しく保つ

## モーション

- `prefers-reduced-motion: reduce` を尊重する
- 必要な場合のみアニメーションを使う
- 点滅は3回/秒を超えない

## テスト基準

新しいコンポーネントは以下を確認してからマージする:

- [ ] キーボードのみで操作できる
- [ ] スクリーンリーダーで適切に読み上げられる
- [ ] カラーコントラスト比が基準を満たす
- [ ] `prefers-reduced-motion` に対応している
