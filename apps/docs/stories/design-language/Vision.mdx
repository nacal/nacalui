# Vision

nacalui は、個人プロジェクト全般で一貫したユーザー体験を提供するためのデザインシステムです。

## 目的

- **プロジェクト間の一貫性** — 複数の個人プロジェクトで同じデザイン言語を共有し、統一感のある体験を実現する
- **開発の加速** — 基盤となるトークンとコンポーネントを再利用することで、UIの意思決定コストを下げる
- **品質の底上げ** — アクセシビリティやレスポンシブ対応を仕組みとして組み込み、すべてのプロジェクトで一定の品質を担保する

## このデザインシステムが目指すもの

- **穏やかで落ち着いたビジュアル** — 派手で目を引くデザインではない
- **アクセシブルで誰でも使える** — 特定の環境に依存しない
- **少ないコンポーネントで多くをカバー** — 網羅的なコンポーネントカタログではない
- **自分のプロジェクトに最適化** — 汎用的なUIフレームワークではない

## 対象

自分自身のWebアプリケーション、ポートフォリオ、ツール、実験的プロジェクトなど、個人で開発するすべてのプロダクト。
